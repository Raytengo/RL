这是我们的小组作业的仓库，用来实现一个RL算法完成扫雷小游戏。
我们打算实现track a的部分，实现一个game playing with q-learning（也可能是别的算法）
我们小组三个人，我们将代码分为三个部分分工合作：游戏建模、算法实现、训练及评估。
1. 游戏建模：
   获取扫雷游戏的数据，并实现两个方式进行游玩。一个是人工游玩，通过在cli输入坐标进行扫雷。一个是用于机器训练的版本。 
   这一个部分实现的是，和算法实现的部分进行交互，获取机器反馈的扫雷坐标，返回一个环境状态。包括{gtid_size, map, end_or_not（可以是list或者dict都可）, reward_for_current_move, trail_count}

2. 算法实现：
   这个部分设计一个RL算法，和环境一直交互，训练模型。

3. 训练及评估：
   这个部分利用1和2，训练一个RL模型，然后在测试集上进行评估，得到包括成功率、最少步数等指标。
